## 4 · Decisiones de arquitectura tomadas

Para entender el porqué de cada elección:

| Decisión | Por qué |
|---|---|
| Astro estático (no SSR) | El sitio es contenido + conversión, no app. SSR no aporta |
| Tailwind v4 vía PostCSS | El plugin Vite de Tailwind no es compatible con rolldown-vite de Astro 6 |
| Override Vite a v7 | `@tailwindcss/postcss` traía Vite 8, Astro 6 pide Vite 7 |
| Form de contacto sin backend | v1 abre WhatsApp pre-rellenado. Cero infra. Cuando haya endpoint, swap del handler |
| Belgiano Serif → Cormorant Garamond | Belgiano es licencia paga (Adobe Fonts). Cormorant es Google Fonts y tiene carácter casi idéntico |
| Sin cookie banner | Plausible no usa cookies. Cuando GTM active GA4, revisar |
| Tarifa Fundadores en vez de "% off" | Protege posicionamiento premium. Escasez (cupo) en vez de oferta |
| Audiencia jerarquizada | Psi-clínico al frente, bienestar legítimo pero secundario |
| Accesibilidad honesta | Los 2 escalones se comunican como cuidado, no se ocultan |
| Buscador demo con mocks | Hook visual fuerte sin endpoint público. Migrable a real cuando exista |
