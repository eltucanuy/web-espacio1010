## 8. Gotchas técnicos para futuras sesiones

- **NO reescribir archivos con acentos vía PowerShell** (`Get-Content |
  Set-Content` rompe el encoding → mojibake). Pasó con nichos.ts; se restauró
  de git y se rehizo con el editor. Usar siempre las herramientas de edición.
- Scripts nuevos en páginas con view transitions: patrón `astro:page-load`
  (ResenasGoogle.astro es el ejemplo de referencia con estado module-level).
- WebFetch de producción puede servir HTML viejo unos segundos post-deploy:
  verificar el deployment READY en Vercel y refetchear con query param nuevo.
- La verdad de datos duros sigue siendo `VERDAD_APP_2026_06_10.md` + la DB;
  las sillas en multiuso (2026-07-05) ya están reflejadas en ambas.
